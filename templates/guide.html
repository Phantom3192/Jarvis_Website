🤖 Talking to {{ bot_name }}

Just chat — no setup required.

You can @mention {{ bot_name }} or DM it directly, or use /chat if you prefer a slash command.

{{ bot_name }} remembers your last couple of messages so it can follow a conversation, and it can also remember things about you long-term — your name, where you're from, what you like — so you don't have to repeat yourself every time. Just tell it naturally, e.g. "call me Alex" or "remember that I love hiking." You can also send images — {{ bot_name }} can look at and describe or answer questions about pictures, depending on the model you've selected.

/setmodelChoose which AI model replies to you — there's a fast one, a smart one, and one that can read images
/mymodelCheck which model you're currently using
/mymemorySee everything {{ bot_name }} remembers about you
/forgetmeWipe everything {{ bot_name }} remembers about you
/clearhistoryClear your recent conversation
/nicknameTell {{ bot_name }} what to call you
/summaryGet a quick summary of your recent conversation
/dndToggle Do Not Disturb — quiets most responses to you
/remindmeSet a DM reminder for later
/myremindersView your active reminders

Daily limit: everyone gets 100 free AI messages a day, resetting at midnight UTC. You'll get a heads-up once you've used 80. If you run out, you can either wait for the reset or spend Jarvis Credits to reset your limit early with /mylimit.

🎮 Games

Chess, Mafia, Hangman, and a server-wide counting game.

Chess

Challenge someone with /chess. Make moves with !move, offer or accept a draw with !draw, resign with !resign, or request to undo your last move with !undo — your opponent has to agree. Stuck? Use !hint for an AI-suggested move; your first hint each turn is free, extra ones cost JC. !chessboard redraws the current board if you've lost track of it.

Mafia 🎭

A full social-deduction game for a group. Start a lobby with /mafia, and the host can pick from 11 possible roles — Mafia, Detective, Doctor, Villager, Vigilante, Jester, Mayor, Serial Killer, Bodyguard, Spy, Escort. Day and night phases move forward automatically once everyone's acted, or when the timer runs out. You can write a "last will" that gets revealed if you're eliminated, and Mafia members get a private chat to coordinate. Run /mafguide for the full rules explainer before your first game.

Hangman

Quick and simple — /hangman starts a round, guess letters to save your character before the gallows fill in. !stophangman ends it early.

Counting Game

A shared server game — take turns counting up one number at a time, and nobody can count twice in a row. Ask an admin to set up a counting channel with !countsetup. If you accidentally break the streak, you can pay JC to save it instead of starting over.

📜 The Rules

A few things to keep in mind.

  • Don't spam commands — flooding {{ bot_name }} with too many commands too quickly will get you temporarily timed out from using the bot.
  • There's a short cooldown between commands, so give it a second between messages.
  • Being banned from {{ bot_name }} (globally or just on this server) means none of its commands will work for you — you'll get a one-time DM letting you know if this happens.
  • Full command list any time: run /help for an interactive, categorized menu.
